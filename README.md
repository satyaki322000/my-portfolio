# Satyaki Portfolio

An Angular 19 + SCSS + Tailwind portfolio built to present Satyaki Banerjee as a senior frontend / full stack engineer with strong HealthTech, GovTech, cloud, and enterprise delivery experience.

## Stack

- Angular 19
- SCSS
- Tailwind CSS v4
- GitHub Pages workflow for free static deployment

## Local development

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm start
```

## Production build

Standard build:

```bash
npm run build
```

GitHub Pages-friendly build:

```bash
npm run build:github-pages
```

## Free deployment

This repository includes `.github/workflows/deploy.yml` for GitHub Pages.

To publish it:

1. Push the project to GitHub.
2. Rename your default branch to `main` if needed.
3. In GitHub, open `Settings -> Pages`.
4. Set the source to `GitHub Actions`.
5. Push to `main` and let the workflow deploy automatically.

For a public repository, GitHub Pages is the cleanest free option for long-term static hosting.
